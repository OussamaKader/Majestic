# Majestic

## Description du projet
Ce projet est une **plateforme e-commerce** développée principalement avec **HTML, CSS et JavaScript** pour la partie front-end. L'objectif est de créer une interface utilisateur fonctionnelle et attrayante permettant aux clients de parcourir les produits, de les ajouter au panier et de simuler un processus de commande.

> **Remarque :** La partie back-end n'est pas incluse dans ce projet front-end, mais elle a déjà été réalisée séparément avec **Laravel**.

## Technologies utilisées
- **HTML** : Structure des pages et contenu.
- **CSS** : Mise en forme, styles et animations.
- **JavaScript** : Interaction dynamique avec l'utilisateur (ajout au panier, calcul du total, validation des formulaires, etc.).
- **Laravel (back-end)** : Gestion des données et logique serveur (non inclus ici).

## Fonctionnalités principales
- Affichage des produits avec image, description et prix.
- Ajout et suppression de produits dans le panier.
- Mise à jour dynamique du total de la commande.
- Interface responsive adaptée aux différents types d’écrans.
